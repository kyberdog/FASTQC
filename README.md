# FASTQC
# Анализатор FASTQ файлов

Программа для анализа FASTQ файлов с графическим интерфейсом, предоставляющая базовую статистику о последовательностях ДНК/РНК.

## Возможности

- **Базовая статистика**:
  - Количество последовательностей
  - Средняя, минимальная и максимальная длина последовательностей
  - Общая длина всех последовательностей
  - Информация о размере файла

- **Анализ качества**:
  - Среднее качество последовательностей
  - Минимальное и максимальное качество
  - Общее количество оценок качества

- **Состав оснований**:
  - Подсчет количества оснований A, T, G, C
  - Процентное соотношение каждого основания

## Требования

Для работы программы необходимо установить следующие зависимости:

```bash
pip install pillow
```
## Установка и запуск

Скачайте файл analiz_FASTQC.py

Установите зависимости (см. выше)

Запустите программу:

```bash
python analiz_FASTQC.py
```
## Использование

Нажмите кнопку "выбрать FASTQ файл" для выбора файла

Поддерживаются форматы:

Обычные FASTQ файлы (.fastq, .fq)

Сжатые GZIP файлы (.fastq.gz, .fq.gz)

Нажмите кнопку "проанализировать" для запуска анализа

Просмотрите результаты в трех разделах:

Базовая статистика

Качество последовательностей

Состав оснований

## Особенности интерфейса

Адаптивный фон - изображение автоматически масштабируется при изменении размера окна

Современный дизайн - цветовая схема гармонирует с фоновым изображением

Удобная навигация - интуитивно понятное расположение элементов

Подробная статистика - вся информация представлена в структурированном виде

## Формат FASTQ

Программа обрабатывает файлы в стандартном формате FASTQ, где каждая последовательность представлена 4 строками:

Идентификатор (начинается с @)

Последовательность нуклеотидов

Разделитель (обычно +)

Строка качества

## Обработка ошибок

Программа включает обработку следующих ситуаций:

Ошибки чтения файлов

Некорректный формат FASTQ

Проблемы с кодировкой

Отсутствие выбранного файла

## Технические детали
Написано на Python 3.x

Поддерживает многопоточность для обработки больших файлов

Оптимизировано для работы с файлами большого размера
